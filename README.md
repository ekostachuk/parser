# CyberLeninka Scraper For PyCharm

Небольшое desktop-приложение на Tkinter для поиска и выгрузки статей с CyberLeninka.

Что умеет:

- искать статьи по ключевому слову;
- исключать статьи по словам-исключениям;
- фильтровать по диапазону лет;
- проверять до заданного количества кандидатов;
- сохранять результаты в `XLSX`, `CSV` и `JSON`;
- показывать лог обработки прямо в окне приложения.

## Запуск в PyCharm

1. Откройте папку проекта `cyberleninka_pycharm_project`.
2. Создайте виртуальное окружение Python 3.11+.
3. Установите зависимости:

```bash
pip install -r requirements.txt
```

4. Запустите файл:

```bash
python main.py
```

## Запуск в браузере

Если удобнее работать через браузер, можно поднять локальную web-версию:

```bash
pip install -r requirements.txt
python web_main.py
```

После запуска откройте локальный адрес:

```text
http://127.0.0.1:8000
```

По умолчанию браузерная версия теперь работает на `Gradio`. Это даёт:

- более удобную форму без ручной HTML-вёрстки;
- живой лог выполнения;
- превью результатов в таблице;
- кнопки скачивания файлов прямо в интерфейсе.

## Поля в интерфейсе

- `Ключевое слово` — обязательное поле.
- `Слова-исключения` — через запятую, например: `туризм, медицина, персонал`.
- `Год от` / `Год до` — в формате `YYYY`.
- `Сколько кандидатов проверить` — например `100`.
- `Папка для сохранения` — куда складывать результаты.

## Важные детали

- Для ускорения сначала используется поисковая выдача `ddgs`, потом открываются только отобранные ссылки.
- Если поисковая выдача не помогла, включается запасной поиск по внутреннему поиску CyberLeninka.
- PDF-ссылки исключаются из кандидатов.
- В `JSON` и `CSV` сохраняется полный текст.
- В `XLSX` текст автоматически очищается от недопустимых символов и укорачивается до лимита Excel.
